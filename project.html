This project aims to create a map with all the information that may be useful to you around where you live. It is currently used by me around La Roque-d'Anthéron, France, but you may clone its code and install it on your website to use it for you and your neighbours' needs.

The site is relying on open-source frameworks such as : OpenLayers and OpenStreetMap. The code is freely available on Github, thanks to julienb for having shared his initial code, used in the Craft Beer Map Project.

Cheers!
